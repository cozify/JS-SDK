# cozify-sdk

> JavaScript SDK for Cozify REST API

[![NPM](https://img.shields.io/npm/v/cozify-sdk.svg)](https://www.npmjs.com/package/cozify-sdk) [![JavaScript Style Guide](https://img.shields.io/badge/code_style-standard-brightgreen.svg)](https://standardjs.com)

## Install

```bash
npm install --save cozify-sdk
```

## Usage

```jsx
import React, { Component } from 'react'

import MyComponent from 'cozify-sdk'

class Example extends Component {
  render () {
    return (
      <MyComponent />
    )
  }
}
```
